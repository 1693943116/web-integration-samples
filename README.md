# MyScript Web Integration Samples

The easy way to integrate mathematical expressions handwriting recognition in a Word add-in.

![](https://myscript.github.io/officeplugins/word-addin.gif)

You will find the complete version of this add-in in app directory.

## How to create your own handwriting recognition Word add-in ?

You will find a quick tutorial [here](http://addresseDuTuto) to create your own Word add-in that use myscript-math-web.

The aim is to show you how is it easy to add handwriting recognition with our web component.

The tutorial directory contains a full sample Word add-in. You must arrive at this result after following our tutorial. 

## Documentation 

The API Reference of myscript-math-web component used for the tutorial is available here: [https://myscript.github.io/myscript-math-web/](https://myscript.github.io/myscript-math-web/) 

## Contribute

If you wish to contribute to myscript-math-web, feel free to fork it, fire `bower install` and use [polyserve](https://github.com/PolymerLabs/polyserve) as an HTTP server. The demo page (in `demo` directory) can help you make your first experimentation.

Please sign our [Contributor License Agreement](CONTRIBUTING.md) before submitting your pull request.
